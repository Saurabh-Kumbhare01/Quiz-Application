# Quiz Application

A responsive React quiz application with category and difficulty selection, timed questions, instant validation, result grading, review mode, theme toggle, sound effects, and local result history.

## Tech Stack

- React
- Vite
- JavaScript
- CSS

## Local Development

Install dependencies:

```powershell
npm install
```

Start the dev server:

```powershell
npm run dev
```

## Production Build

```powershell
npm run build
npm run preview
```

The optimized deployment output is generated in `dist/`.

## Deployment

This project is ready for Vercel, Netlify, Render, and other static hosting platforms. The included `vercel.json` and `netlify.toml` point deployments to the Vite production build.
